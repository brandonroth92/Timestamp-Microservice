API Project: Timestamp Microservice

By Brandon Roth

What is this?

This is an API project for freeCodeCamp's Back End Development Certification that will accept either a human readable date or a UNIX timestamp and return a JSON object with both formats. If the input does not conform to either format, it will return null for both values.

Usage:

/December%2015,%202015
/1450137600

Example Output:

{ "unix": 1450137600, "natural": "December 15, 2015" }
{ "unix": null, "natural": null }
